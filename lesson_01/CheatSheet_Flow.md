# JavaScript Flow Control Cheat Sheet

Conditional Execution
=====================

if (expression)  // if expression is true
  statement;     // execute statement

if (expression)  // if expression is true
  statement;     // execute statement
else             // otherwise
  statement;     // execute statement

if (expression)      // if expression is true
  statement;         // execute statement
else if (expression) // otherwise if expression is true
  statement;         // execute statement
